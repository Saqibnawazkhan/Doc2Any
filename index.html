Help & Support

Find answers to common questions

What file formats are supported?

We support over 50 file formats including PDF, DOCX, DOC, ODT, TXT, RTF, XLSX, PPTX, HTML, CSV, and many more. You can convert between any of these formats seamlessly.

Is my data secure?

Absolutely! All file transfers are encrypted using SSL/TLS. Your files are automatically deleted from our servers within 1 hour after conversion. We never share or access your file contents.

What is the maximum file size?

Free users can convert files up to 25 MB. Pro users get 100 MB limit, and Enterprise users can convert files up to 500 MB. Need more? Contact us for custom solutions.

How do I get API access?

API access is available with Pro and Enterprise plans. After signing up, you'll receive your API key in your dashboard. Check our documentation for integration guides and examples.

Can I use Doc2Any offline?

Yes! Our web app uses client-side processing for many conversions, so once loaded, you can convert files even without an internet connection. Some advanced features require server processing.
